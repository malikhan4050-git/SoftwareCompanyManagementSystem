Software Company Management System

Project Overview
This is a Software Company Management System developed in Java which demonstrates the practical use of multiple data structures to manage employee records efficiently. The system is designed to handle typical company operations such as adding, searching, updating, and deleting employee records, processing priority-based tasks, generating reports, and supporting undo/redo functionality.

This project is menu-driven and beginner-friendly, making it easy to understand and extend.

---

Features / Core Functionalities

1. Add a New Employee Record  
   - Adds employee details such as ID, name, designation, salary, and priority level.

2. Search for an Employee  
   - Efficiently searchs for employees by their ID using a Binary Search Tree (BST).

3. Update or Delete an Employee Record
   - Updates employee details or remove a record from the system.

4. Display All Employees  
   - Displays all employee records in sorted order by their ID.

5. Generate Reports  
   - Shows total number of employees currently in the system.

6. Priority-Based Task Processing  
   - Processes employees or tasks based on priority using a Max Heap.

7. Undo / Redo Last Operations
   - Tracks operations using a Stack and allow undo/redo functionality.

---

Data Structures Used

Binary Search Tree (BST) – for storing and searching employee records efficiently

Max Heap – for processing priority-based tasks

Stack – for undo/redo operations
